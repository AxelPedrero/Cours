Les deux types de balise

Il existe deux types de balises :

  1. Les balises usuelles encadrent du code <body> et </body>

  2. Les balises dites orphelines n'encadrent aucun code, elles se ferment elles mêmes. Par exemple <br/> abréviation de break row;

  3. Les balises ditent orphelines n'encadrent aucun code, elles se ferment elles mêmes. Par exemple <hr/> abréviation de horizontal row;
